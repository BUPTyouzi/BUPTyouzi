# 职业发展活动介绍
每周固定时间举行一次校友间的分享和讨论，主要与年轻校友讨论职业发展相关内容。
### 主题
不限主题，由嘉宾自拟，可关于职业、人文、技术等。
### 形式
茶话会形式，由一人分享后就分享主题多人讨论。
### 流程
1.听众自我介绍（9:00-9:15北京 /18:00-18:15PST）

2.嘉宾分享（约30-40min）

3.QA+讨论 （没有固定时间，一般1-2个小时，建议到没问题。）
### 特点
1.形式与时间自由，不限制分享时间；

2.无拘束，不录屏不回放；

3.校友讨论活跃
